# Platforms and Delivery

## Style Recommendations

| Platform | Recommended styles | Composition note |
| --- | --- | --- |
| Xiaohongshu | 1 Soft 3D, 2 Clay Figure, 5 Anime, 8 Textured Paint | Warm expression, bright face, simple background |
| X | 1 Soft 3D, 4 Editorial 2D, 6 Comic Ink, 7 Pixel Art | Strong silhouette and contrast at small size |
| GitHub | 3 Flat Vector, 6 Comic Ink, 7 Pixel Art | Distinct outline, minimal detail, no tiny accessories |
| General personal brand | 1 Soft 3D, 3 Flat Vector, 4 Editorial 2D | Neutral background and broad cross-platform usability |

## Master and Export Sizes

Generate and preserve a high-resolution square master. Export only when requested.

| Deliverable | Size | Notes |
| --- | ---: | --- |
| Master square | 2048 x 2048 px preferred | Keep as the editing source |
| General social avatar | 1024 x 1024 px | Default shareable output |
| X profile image | 400 x 400 px | Design for circular display |
| GitHub profile image | 500 x 500 px | Keep important details away from corners |
| Xiaohongshu profile image | 400 x 400 px | Design for circular display |

Platform interfaces can change. Treat these as practical export presets, not upload-limit guarantees.

## Circular Preview

- Keep the square master unchanged.
- Create the circular preview as a separate PNG.
- Center the eyes slightly above the vertical midpoint.
- Keep hair, chin, and distinctive features inside the central 80% circle.
- Do not use a visible circular border unless requested.

## File Naming

Use descriptive ASCII names:

- `avatar-master.png`
- `avatar-circle-preview.png`
- `avatar-x-400.png`
- `avatar-github-500.png`
- `avatar-xiaohongshu-400.png`
- `avatar-prompt.md`

If a file exists, add a version suffix rather than overwriting it.
